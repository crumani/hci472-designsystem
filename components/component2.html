Buttons

Buttons are used to take action on a webpage.

Default

This is the primary button for taking general actions.

Example:

Code:



      <button class="btn-default">Default</button>

Call-to-action

This buttons is used for main actions, such as a call-to-action. Use this only once on a singular page. It can also be used alongside a default button to show two options.

Example:

Code:



      <button type="button" class="btn btn-cta">Call-to-action <svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" fill="currentColor" class="bi bi-chevron-right text-center" viewBox="0 0 16 16">
        <path fill-rule="evenodd" d="M4.646 1.646a.5.5 0 0 1 .708 0l6 6a.5.5 0 0 1 0 .708l-6 6a.5.5 0 0 1-.708-.708L10.293 8 4.646 2.354a.5.5 0 0 1 0-.708"/>
      </svg> </button>

Accessibility

Ensure contrast adheres to WCAG 2.1 Guidelines with necessary contrast of the background and foreground. Using ARIA attributes enables human-centered experiences with screenreaders and other assistive technology used by those with disabilities. Make sure the ARIA descriptions are understandable and clear enough for a wide audience.
